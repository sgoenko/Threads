Разработать многопоточное приложение.
Использовать возможности, предоставляемые пакетом java.util.concurrent.
Не использовать synchronized.
Все сущности, желающие получить доступ к ресурсу, должны быть потоками.
Не использовать графический интерфейс. Приложение должно быть консольным

Порт. 
Корабли заходят в порт для разгрузки/загрузки контейнеров. 
Число контейнеров, находящихся в текущий момент в порту и на корабле,
должно быть неотрицательным и не превышать заданную грузоподъемность 
судна и вместимость порта. В порту работает несколько причалов. У одного
причала может стоять один корабль. Корабль может загружаться у причала,
разгружаться или выполнять оба действия.
